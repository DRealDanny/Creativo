# 🛑 CUSTOM PROJECT RULES FOR DANNY'S CMS

1. **NO TAILWIND CSS:** Do NOT use Tailwind CSS utility classes under any circumstances. We strictly use Vanilla CSS Modules (e.g., `page.module.css`) for all styling. Keep the JSX clean.
2. **DESIGN PHILOSOPHY:** Prioritize clean, minimalist, intentional UI/UX layout. Use logical spacing and highly readable structures.
3. **TECH STACK:** This is a Next.js App Router project acting as a Headless CMS.
4. **AUTHENTICATION:** We will be using NextAuth (Auth.js) for Google Sign-In authentication.
